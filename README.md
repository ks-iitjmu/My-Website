## 🔧 Installation & Setup

To run this project locally, follow these steps:

```bash
# Clone this repository
https://github.com/ks-iitjmu/My-Website.git

# Navigate to the project directory
cd My-Website

# Install dependencies
npm install

# Start the development server
npm start
```
